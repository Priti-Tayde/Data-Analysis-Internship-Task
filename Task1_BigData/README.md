# Task 1: Big Data Analysis using PySpark

## 📌 Objective
To perform scalable data analysis on a large dataset using PySpark.

## 📊 Dataset
Netflix Movies and TV Shows Dataset

## 🔧 Tools Used
- PySpark
- Python
- Matplotlib

## ⚙️ Steps Performed
- Data loading using PySpark
- Data preprocessing (handling null values, duplicates)
- Exploratory Data Analysis (EDA)
- Data visualization

## 📈 Key Insights
- Movies are more than TV Shows on Netflix
- USA has the highest content
- Content increased rapidly after 2015
- TV-MA is the most common rating
