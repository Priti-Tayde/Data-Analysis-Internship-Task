# Task 2: Predictive Analysis using Machine Learning

## 📌 Objective
To build a machine learning model to predict student performance.

## 📊 Dataset
Students Performance Dataset

## 🔧 Tools Used
- Python
- Pandas
- Matplotlib
- Scikit-learn

## ⚙️ Steps Performed
- Data preprocessing (cleaning, handling duplicates)
- Exploratory Data Analysis (EDA)
- Feature selection
- Model training using Linear Regression
- Model evaluation

## 📈 Key Insights
- Strong correlation between reading, writing, and math scores
- Students with better reading and writing scores perform better in math
- The model predicts math scores effectively
