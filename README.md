Data Analysis Internship - CODTECH


This repository contains all the tasks completed during the Data Analysis Internship. Each task demonstrates different skills including big data processing, machine learning, data visualization, and natural language processing.



🛠 Tools & Technologies Used

*Python
*Google Colab
*PySpark
*Pandas, NumPy
*Matplotlib
*Scikit-learn
*Power BI
*NLP (TextBlob)



📂 Tasks Overview


🔹 Task 1: Big Data Analysis using PySpark
Performed analysis on a large dataset (Netflix Movies and TV Shows) using PySpark to demonstrate scalability and extract meaningful insights.


🔹 Task 2: Predictive Analysis using Machine Learning
Built a regression model to predict student performance (math scores) based on reading and writing scores.


🔹 Task 3: Dashboard Development
Created an interactive dashboard using Power BI to visualize data and generate actionable insights.


🔹 Task 4: Sentiment Analysis
Performed sentiment analysis on textual data using NLP techniques to classify reviews as positive, negative, or neutral.
